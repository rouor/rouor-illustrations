# Prompt Examples

## 只做配图规划

```text
Use $rourou-illustrations 先不要生图。
请分析下面这篇文章哪里值得配图，输出 5 张左右的 shot list。
每张图写清楚：放置段落、主题、核心意思、结构类型、ROUROU 的核心动作、纸本道具和中文标注词。

<粘贴文章>
```

## 生成一组正文配图

```text
Use $rourou-illustrations 把下面这篇文章生成 4 张正文配图。
要求：16:9 横版、纯白背景、暖白纸本 ROUROU、手绘墨线、少量莫兰迪色。
每张图只讲一个核心结构，不要做 PPT 信息图。

<粘贴文章>
```

## 长文配图策略

```text
Use $rourou-illustrations 给这篇长文做配图策略。
不要平均配图，只挑核心判断、输入输出闭环、前后对比、常见坑和角色状态变化。
先输出 6-8 张 shot list，不要生成图片。

<粘贴文章>
```

## 单个观点

```text
Use $rourou-illustrations 为这个观点生成一张 16:9 正文配图：

先验证最小闭环，再扩大投入。

让 ROUROU 承担核心动作，重新发明一个纸本物理隐喻。中文标注不超过 4 处。
```

## 工作流主题

```text
Use $rourou-illustrations 为“把零散信息加工成可执行判断”生成一张正文配图。
不要画正式流程图；让 ROUROU 通过折叠、分类或拼接完成核心动作。
```

## 改图：修复角色走样

```text
Use $rourou-illustrations 修复这张图里的 ROUROU。
严格参考角色设定图，恢复暖白揉皱纸袋身体、蓝色回形针、红色斜缝线、直腿和弹簧腿。
不要改变核心构图和文字。
```

## 改图：清理伪角色

```text
Use $rourou-illustrations 清理图里的伪 ROUROU。
整张图只保留一个 ROUROU，蓝色回形针只能出现在角色身上。
普通纸片改用胶带、折口、插槽或黑色长尾夹连接。
```

## 改图：增加少量莫兰迪色

```text
Use $rourou-illustrations 保持构图和角色不变，只增加少量鼠尾草绿、灰芥末黄或灰紫纸片。
彩色面积控制在 18% 以下，不要给背景铺色。
```
